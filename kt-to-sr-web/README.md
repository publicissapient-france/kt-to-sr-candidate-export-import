# kt-to-sr-web

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploy
```shell script
npm run deploy
```
> Needs AWS CLI and BUCKET_NAME environment variable.
